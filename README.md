# Placement Management System (production-grade foundation)

Database-driven full-stack platform for end-to-end campus placement lifecycle management across **students**, **companies**, and **placement cell admins**.

## Tech stack

- **Frontend**: React + Vite + React Router + TanStack Query + Tailwind
- **Backend**: Flask (cookie-session auth) + MySQL (raw SQL)
